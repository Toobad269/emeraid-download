Projekte / Roadmap.md

Was als Nächstes ansteht — kurz festgehalten und von der KI sortiert.

• Recherche: 3 Quellen sammeln
• Entwurf schreiben & zusammenfassen
• Review + veröffentlichen
▸ KI: „Fasse meine offenen Aufgaben zusammen" → 4 Notizen gelesen ✓

Alles drin — lokal und privat

Kein Abo, keine Cloud-Pflicht. Deine Notizen sind normale Markdown-Dateien auf deinem Mac.

✦

KI-Agent

Liest, durchsucht und schreibt direkt in deinem Vault — mit deinem eigenen Schlüssel oder lokalem Ollama.

⚙

Workflow-Baukasten

Verkette KI-Schritte visuell: Web-Suche, Notizen, Bedingungen, Schleifen — sogar zeitgesteuert.

◎

Multi-Agent-Modus

Eine ganze KI-Crew (CEO → Teams → Worker) arbeitet selbstständig an deinem Ziel.

🜨

Graph & Code

Sieh deine Verlinkungen als Graph, bearbeite Code mit Ausführen-Knopf, springe per ⌘K überallhin.

⌘

Lokale API

Steuere Emeraid aus eigenen Skripten — ein kleiner Server auf deinem PC, token-geschützt.

🔒

Privat by Design

Keine Anmeldung, keine Tracker. Deine Daten verlassen deinen Mac nur, wenn du es willst.

↓ Für macOS herunterladen
Erster Start: Da die App noch nicht bei Apple signiert ist, zeigt macOS evtl. „nicht überprüft". Dann einmal Rechtsklick auf Emeraid.app → Öffnen → Öffnen (oder unter Systemeinstellungen → Datenschutz & Sicherheit → „Trotzdem öffnen").
